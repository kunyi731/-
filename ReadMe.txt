頑張って作ったモンハンシミュレータ for MH4 ver.0.9
2013/10/5

■説明
ニンテンドー３ＤＳ専用ソフト「モンスターハンター４」
のゲーム内において、発動スキルに対応する防具の組み合わせを検索する
「スキルシミュレータ」「スキルサーチ」と呼ばれるツールの一つです。

■動作環境
Java Runtime Environment Version 6　以上
OSは、Javaが動けば多分何でもOKだと思います。

■使い方
頑シミュ.jarをダブルクリックして下さい。
Macの方、環境が良く分からない方などは下記のサイトを確認下さい。
最新版についてもこちらで配布しています。リンクフリーです。
http://www.geocities.jp/masax_mh/

旧バージョンの保存データを引き継ぐ場合、
旧バージョンのsaveフォルダを新しくダウンロードしたフォルダ内にコピーして下さい。
